# ECG Viewer - ADS1294R 调试工具

一款基于 Python + PyQt5 的专业 ECG (心电图) 波形显示和分析软件，专为 ADS1294R 生理信号采集芯片设计。

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![PyQt5](https://img.shields.io/badge/PyQt5-5.15+-green.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

---

## 📋 功能特性

### 核心功能
- **4 通道实时波形显示** - 支持 ADS1294R 的 4 个通道同步显示
- **串口通信** - 支持 9600-921600 波特率，自动数据解析
- **24 位数据解析** - 精确解析 ADS1294R 输出的 24 位二进制补码数据
- **实时 FFT 频谱分析** - 每通道独立的频谱显示 (0-250Hz)

### 高级功能
- **心率检测** - 自动 R 波检测，实时 BPM 显示
- **呼吸率检测** - 从呼吸阻抗通道提取呼吸波形，计算 RPM
- **数字滤波** - 50Hz 陷波、0.5-150Hz 带通滤波
- **数据保存** - CSV 格式导出，支持元数据
- **截图功能** - 一键保存当前波形图

### 界面特性
- **暗色主题** - 专业医疗仪器风格，长时间使用不疲劳
- **可调节时基** - 支持 1-10 秒/格 缩放
- **通道控制** - 独立开关和增益调节
- **实时状态** - 显示连接状态、数据统计、生理参数

## 🚀 快速开始

### 方法一：使用可执行文件 (推荐)

1. 下载 `ECG Viewer.exe`
2. 双击运行
3. 选择串口和波特率
4. 点击"连接"开始采集

### 方法二：从源代码运行

#### 系统要求
- Python 3.8 或更高版本
- Windows 7/10/11

#### 安装步骤

1. **克隆或下载项目**
   ```bash
   cd F:\FE\ecg_viewer
   ```

2. **创建虚拟环境** (可选但推荐)
   ```bash
   python -m venv venv
   venv\Scripts\activate
   ```

3. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

4. **运行程序**
   ```bash
   python main.py
   ```

## 🔌 硬件连接

### ADS1294R 引脚连接

| ADS1294R | STM32F103 | 说明 |
|----------|-----------|------|
| DIN | PA7 (SPI1_MOSI) | SPI 数据输入 |
| DOUT | PA6 (SPI1_MISO) | SPI 数据输出 |
| SCLK | PA5 (SPI1_SCK) | SPI 时钟 |
| CS | PA4 | 片选 |
| DRDY | PC8 | 数据就绪中断 |
| START | PC9 | 开始转换 |
| RESET | PC10 | 复位 |
| AVDD | 3.3V | 电源 |
| DGND | GND | 地 |

### 推荐串口配置

| 参数 | 值 |
|------|-----|
| 波特率 | 115200 或更高 |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验位 | None |

## 📊 通信协议

### 数据帧格式 (57 字节)

```
┌──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────┬──────────┬─────┐
│ STX0 │ STX1 │LEN_L │LEN_H │ SRC  │ DST  │ SEQ  │MSGID │  DATA   │ CRC │
│ 0x55 │ 0xAA │ 0x30 │ 0x00 │ 0x01 │ 0x00 │ 0xXX │ 0x20 │ 48 字节  │ XX  │
└──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────┴──────────┴─────┘
```

### DATA 载荷 (48 字节)

- 10 个样本 × 4 通道 × 3 字节/通道
- 每通道 24 位数据，MSB 优先，二进制补码格式
- 数据范围：-8,388,608 ~ +8,388,607

### CRC8 算法

- 多项式：0x07
- 初始值：0x00

## 🎛️ 使用说明

### 基本操作

1. **连接设备**
   - 选择正确的 COM 端口
   - 设置波特率 (与 MCU 固件一致)
   - 点击"连接"按钮

2. **查看波形**
   - 4 个通道自动显示
   - 绿色表示心率正常，红色表示异常
   - 可滚动鼠标缩放 Y 轴

3. **保存数据**
   - 点击"保存 CSV"选择保存路径
   - 数据包含时间戳和通道标签

4. **截图**
   - 点击"截图"按钮
   - 选择保存位置

### 高级功能

- **暂停/继续** - 冻结当前波形显示
- **滤波开关** - 启用/禁用数字滤波
- **自动缩放** - 自动调整 Y 轴范围
- **清除显示** - 清空缓冲区重新开始
