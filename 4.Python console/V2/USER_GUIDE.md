## 简介

ECG Viewer 是一款专业的心电信号（ECG）上位机软件，专为 ADS1294R 硬件设计。它支持实时波形显示、心率/呼吸率监测、数据记录和多种格式导出。

### 主要特性

- ✅ 4 通道实时 ECG 波形显示
- ✅ 心率和呼吸率自动检测
- ✅ FFT 频谱分析
- ✅ 多种数据导出格式（CSV、EDF、MATLAB）
- ✅ 可配置采样率（250/500/1000/2000 SPS）
- ✅ 数字滤波（工频滤波、基线漂移校正）
- ✅ 测试模式（无需硬件即可测试）

## 快速开始

### 步骤 1: 启动程序

```bash
python main.py
```

程序窗口将居中显示在屏幕中央。

### 步骤 2: 配置串口

1. 在串口面板中选择正确的 COM 端口
2. 选择波特率（默认 115200）
3. 选择采样率（默认 500 SPS）
4. 点击"连接"按钮

### 步骤 3: 观察波形

连接成功后，程序将实时显示 4 通道 ECG 波形：
- **CH1**: 呼吸阻抗
- **CH2**: RA（右臂）导联
- **CH3**: LA（左臂）导联
- **CH4**: LL（左腿）导联

### 步骤 4: 查看生理参数

状态栏将实时显示：
- ♥ **心率**: 正常范围 60-100 BPM
- 🫁 **呼吸率**: 正常范围 12-20 RPM
- 📊 **采样率**: 当前设置的采样率

## 功能说明

### 1. 波形显示

- **4 通道独立显示**: 每个通道占用一个绘图区域
- **自动缩放**: 根据信号幅度自动调整 Y 轴范围
- **时间基准**: X 轴显示时间（秒），每格 1 秒

### 2. 控制面板

| 按钮 | 功能 |
|------|------|
| 💾 导出数据 | 打开导出菜单（CSV/EDF/MATLAB/JSON） |
| 📷 截图 | 保存当前窗口截图 |
| ⏸ 暂停 | 暂停/继续波形更新 |
| 🔧 滤波 | 开启/关闭数字滤波器 |
| 📐 自动缩放 | 开启/关闭 Y 轴自动缩放 |
| 🗑 清除 | 清除所有显示数据 |

### 3. 采样率配置

支持的采样率：
- **250 SPS**: 低功耗模式
- **500 SPS**: 标准模式（默认）
- **1000 SPS**: 高分辨率模式
- **2000 SPS**: 超高分辨率模式

> ⚠️ **注意**: 更改采样率后，心率和呼吸率检测算法会自动适应。

### 4. 滤波器

内置数字滤波器包括：
- **50/60 Hz 工频陷波器**: 消除电源干扰
- **高通滤波器**: 去除基线漂移（截止频率 0.5 Hz）
- **低通滤波器**: 去除高频噪声（截止频率 40 Hz）

## 数据导出

### 1. CSV 格式

**用途**: 通用数据格式，可用 Excel 打开

**文件内容**:
- 文件头：采样率、通道数、创建时间
- 数据列：样本序号、时间 (s)、CH1-CH4 值

**打开方式**: Excel、LibreOffice Calc、Python pandas

### 2. EDF 格式（医学标准）

**用途**: 欧洲数据格式，医疗行业标准的生理数据格式

**兼容性**:
- EDFBrowser（免费）
- BioSig
- MATLAB（带 EDF 支持）

**文件内容**:
- 标准 EDF 头信息
- 16 位整数数据
- 所有 4 个通道

### 3. MATLAB 格式 (.mat)

**用途**: MATLAB/Octave 数据分析

**要求**: 需要安装 scipy 库

**加载方法**:
```matlab
data = load('ecg_20240101_120000.mat');
ecg = data.ecg_data;
fs = data.sampling_rate;
```

### 4. JSON 报告

**用途**: 快速查看统计摘要

**包含内容**:
- 记录时长
- 样本数
- 心率和呼吸率
- 各通道的统计信息（最大值、最小值、平均值、标准差）

## 故障排除

### 问题 1: 无法打开程序

**症状**: 双击后无反应或闪退

**解决方案**:
1. 检查 Python 版本（需要 3.8+）
2. 确保虚拟环境已激活
3. 运行 `pip install -r requirements.txt` 重新安装依赖

### 问题 2: 找不到串口

**症状**: 端口下拉列表显示"无可用串口"

**解决方案**:
1. 检查硬件是否已连接
2. 安装串口驱动程序（CH340/CP2102 等）
3. 在设备管理器中确认 COM 口号

### 问题 3: CRC 错误计数增加

**症状**: 状态栏显示 CRC 错误 > 0

**可能原因**:
- 波特率不匹配
- 串口线质量差
- 电磁干扰

**解决方案**:
1. 确认波特率设置为 115200
2. 更换高质量的串口线
3. 降低波特率或增加屏蔽

### 问题 4: 心率/呼吸率显示"--"

**症状**: 波形正常但无生理参数显示

**可能原因**:
- 数据记录时长不足（需要至少 1 秒数据）
- 信号质量差，无法检测
- 滤波器关闭导致干扰

**解决方案**:
1. 等待更多数据积累
2. 开启滤波器
3. 检查电极接触是否良好

### 问题 5: 波形显示异常

**症状**: 波形杂乱或超出显示范围

**解决方案**:
1. 点击"清除"按钮清空缓冲区
2. 检查采样率设置是否与硬件匹配
3. 确认通道增益设置正确

## 附录 A: 通信协议

### 帧格式（57 字节）

```
偏移  大小  字段     说明
0     1     STX0     帧起始 (0x55)
1     1     STX1     帧起始 (0xAA)
2-3   2     LEN      数据长度 (48 字节，小端)
4     1     SRC      源 ID (0x22 = FALCON_FE)
5     1     DST      目标 ID (0x20 = FALCON_HOST)
6     1     SEQ      序列号 (0-255 循环)
7     1     MSGID    消息类型 (0x20 = ADS129X_DATA)
8-55  48    DATA     4 样本 × 4 通道 × 3 字节 (24 位有符号，MSB 优先)
56    1     CRC      CRC8 校验和 (多项式 0x07)
```

### 数据格式

- **采样率**: 500 SPS（默认）
- **帧率**: 125 FPS（500 ÷ 4 样本）
- **分辨率**: 24 位有符号整数
- **通道顺序**: CH1, CH2, CH3, CH4
